Turn off your monitor in C#

Wim Haanstra
This post was created on February 3, 2013.
C#

I once answered a question on Stack Overflow on how to turn your monitor off in C#. It seems that the answer no longer worked on Windows 7 en 8. So here is an updated version, which works for me on at least Windows 7. 

 1 namespace MonitorOff {
 2 
 3 	public enum MonitorState {
 4 		MonitorStateOn = -1,
 5 		MonitorStateOff = 2,
 6 		MonitorStateStandBy = 1
 7 	}
 8 
 9 	public partial class Form1 : Form {
10 		[DllImport("user32.dll")]
11 		private static extern int SendMessage(int hWnd, int hMsg, int wParam, int lParam);
12 
13 		public Form1() {
14 			InitializeComponent();
15 			SystemEvents.SessionSwitch += SystemEvents_SessionSwitch;
16 		}
17 
18 		void SystemEvents_SessionSwitch(object sender, SessionSwitchEventArgs e) {
19 			SetMonitorInState(MonitorState.MonitorStateOff);
20 		}
21 
22 		private void button1_Click(object sender, EventArgs e) {
23 			SetMonitorInState(MonitorState.MonitorStateOff);
24 		}
25 
26 		private void SetMonitorInState(MonitorState state) {
27 			SendMessage(0xFFFF, 0x112, 0xF170, (int)state);
28 		}
29 	}
30 }

As an added bonus I also added some code to turn off your monitor when you lock your machine (need to keep this app running of course).
