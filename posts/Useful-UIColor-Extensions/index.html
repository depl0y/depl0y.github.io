Useful UIColor extension

Wim Haanstra
This post was created on June 22, 2013.
objC UIColor

Just wanted to post this, because I found these UIColor extensions really useful. This extension contains 2 methods, which do exactly what I want them to do:

  1. How do I invert a UIColor
  2. How can I detect contrast from a UIColor

Header file

1 @interface UIColor (Detection)
2 
3 - (UIColor*) invertColor;
4 - (BOOL) isDarkColor;
5 
6 @end

Implementation file

 1 #import "UIColor+Detection.h"
 2 
 3 @implementation UIColor (Detection)
 4 
 5 - (UIColor*) invertColor {
 6 	const CGFloat *componentColors = CGColorGetComponents(self.CGColor);
 7 
 8 	return [[UIColor alloc] initWithRed:(1.0 - componentColors[0])
 9 								  green:(1.0 - componentColors[1])
10 								   blue:(1.0 - componentColors[2])
11 								  alpha:componentColors[3]];
12 }
13 
14 - (BOOL) isDarkColor {
15 	const CGFloat *componentColors = CGColorGetComponents(self.CGColor);
16 
17     CGFloat colorBrightness = ((componentColors[0] * 299) +
18 							   (componentColors[1] * 587) +
19 							   (componentColors[2] * 114)) / 1000;
20 
21 	return (colorBrightness &lt; 0.5);
22 }
23 
24 @end

The detection for color contrast can of course be refined. This just returns a YES or NO, but based on the colorBrightness value you can also make this method return other values. In my case, this would be enough.
