Resizing an image in C#

Wim Haanstra
This post was created on January 26, 2010.
C#

For a project I am working on, I was looking for a good way to resize images in a few different ways. I have done image resizing before, but it really feels that I am re-inventing the wheel every time I do this. So now I decided to write a nice static method which can resize an Image to certain dimensions and outputs an Image object again.

The method requires 3 parameters:

1. An image object. I hope you know how to load a file in an Image object (hint: FromFile).
2. A Resolution object, being nothing more than a small class I created, which also contains the width and the height.
3. The mode you want to use for resizing the image. I explained the modes in the enum listed below.

It probably isnt the neatest code you've ever seen, so if you have any comments, please let me know. Code and more explanation after the break...

  1 ///
  2 /// Method resizes the image so it fits the wanted resolution best.
  3 ///
  4 ///The image to be resized
  5 ///The resolution which the image should be when the method is ready.
  6 ///The mode for resizing the image.
  7 public static System.Drawing.Image Resize(System.Drawing.Image image, Resolution targetResolution, ResizeMode resizeMode)
  8 {
  9 	int sourceWidth = image.Width;
 10 	int sourceHeight = image.Height;
 11 
 12 	int targetWidth = targetResolution.Width;
 13 	int targetHeight = targetResolution.Height;
 14 
 15 	// Supplied image is landscape, while the target resolution is portait OR
 16 	// supplied image is in portait, while the target resolution is in landscape.
 17 	// switch target resolution to match the image.
 18 	if ((sourceWidth &gt; sourceHeight &amp;&amp; targetWidth &lt; targetHeight) || (sourceWidth &lt; sourceHeight &amp;&amp; targetWidth &gt; targetHeight))
 19 	{
 20 		targetWidth = targetResolution.Height;
 21 		targetHeight = targetResolution.Width;
 22 	}
 23 
 24 	float ratio = 0;
 25 	float ratioWidth = ((float)targetWidth / (float)sourceWidth);
 26 	float ratioHeight = ((float)targetHeight / (float)sourceHeight);
 27 
 28 	if (ratioHeight &lt; ratioWidth)
 29 		ratio = ratioHeight;
 30 	else
 31 		ratio = ratioWidth;
 32 
 33 	Bitmap newImage = null;
 34 
 35 	switch (resizeMode)
 36 	{
 37 		case ResizeMode.Normal:
 38 		default:
 39 			{
 40 				int destWidth = (int)(sourceWidth * ratio);
 41 				int destHeight = (int)(sourceHeight * ratio);
 42 
 43 				newImage = new Bitmap(destWidth, destHeight);
 44 				using (Graphics graphics = Graphics.FromImage((System.Drawing.Image)newImage))
 45 				{
 46 					graphics.InterpolationMode = InterpolationMode.HighQualityBicubic;
 47 					graphics.DrawImage(image, 0, 0, destWidth, destHeight);
 48 				}
 49 
 50 				break;
 51 			}
 52 		case ResizeMode.Crop:
 53 			{
 54 				if (ratioHeight &gt; ratioWidth)
 55 					ratio = ratioHeight;
 56 				else
 57 					ratio = ratioWidth;
 58 
 59 				int destWidth = (int)(sourceWidth * ratio);
 60 				int destHeight = (int)(sourceHeight * ratio);
 61 
 62 				newImage = new Bitmap(targetWidth, targetHeight);
 63 
 64 				int startX = 0;
 65 				int startY = 0;
 66 
 67 				if (destWidth &gt; targetWidth)
 68 					startX = 0 - ((destWidth - targetWidth) / 2);
 69 
 70 				if (destHeight &gt; targetHeight)
 71 					startY = 0 - ((destHeight - targetHeight) / 2);
 72 
 73 				using (Graphics graphics = Graphics.FromImage((System.Drawing.Image)newImage))
 74 				{
 75 					graphics.InterpolationMode = InterpolationMode.HighQualityBicubic;
 76 					graphics.DrawImage(image, startX, startY, destWidth, destHeight);
 77 				}
 78 
 79 				break;
 80 			}
 81 		case ResizeMode.Stretch:
 82 			{
 83 				newImage = new Bitmap(targetWidth, targetHeight);
 84 				using (Graphics graphics = Graphics.FromImage((System.Drawing.Image)newImage))
 85 				{
 86 					graphics.InterpolationMode = InterpolationMode.HighQualityBicubic;
 87 					graphics.DrawImage(image, 0, 0, targetWidth, targetHeight);
 88 				}
 89 				break;
 90 			}
 91 		case ResizeMode.Fill:
 92 			{
 93 				newImage = new Bitmap(targetWidth, targetHeight);
 94 
 95 				int destWidth = (int)(sourceWidth * ratio);
 96 				int destHeight = (int)(sourceHeight * ratio);
 97 
 98 				int startX = 0;
 99 				int startY = 0;
100 
101 				if (destWidth &lt; targetWidth)
102 					startX = 0 + ((targetWidth - destWidth) / 2);
103 
104 				if (destHeight &lt; targetHeight)
105 					startY = 0 + ((targetHeight - destHeight) / 2);
106 
107 				newImage = new Bitmap(targetWidth, targetHeight);
108 				using (Graphics graphics = Graphics.FromImage((System.Drawing.Image)newImage))
109 				{
110 					graphics.InterpolationMode = InterpolationMode.HighQualityBicubic;
111 					graphics.DrawImage(image, startX, startY, destWidth, destHeight);
112 				}
113 
114 				break;
115 			}
116 	}
117 
118 	return (System.Drawing.Image)newImage;
119 }

The resizeMode parameter is of the type ResizeMode, which is nothing more than a simple enumeration, like this:

 1 ///
 2 /// Modes available for the resize method of images.
 3 ///
 4 public enum ResizeMode
 5 {
 6 	///
 7 	/// This will resize images to the resolution nearest to the target resolution. Images can become smaller when using this option
 8 	///
 9 	Normal = 1,
10 	///
11 	/// This will stretch an image so it always is the exact dimensions of the target resolution
12 	///
13 	Stretch = 2,
14 	///
15 	/// This will size an image to the exact dimensions of the target resolution, keeping ratio in mind and cropping parts that can't
16 	/// fit in the picture.
17 	///
18 	Crop = 3,
19 	///
20 	/// This will size an image to the exact dimensions of the target resolution, keeping ratio in mind and filling up the image
21 	/// with black bars when some parts remain empty.
22 	///
23 	Fill = 4
24 }

And the targetResolution parameter, is a simple class containing some properties like Name, Width and Height.

This method seems to do everything I need, if you need anything more, or any information about this code, just email me or reply here.
